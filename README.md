# Dexterous Manipulation Simulation

This project simulates dexterous robotic manipulation using geometric and spatial hand-object representations. It explores advanced reinforcement learning techniques for training robotic hands to perform complex, in-hand manipulation tasks within a physics-based environment.

## Key Features
*   Implements the DexRepNet++ architecture for learning manipulation policies.
*   Provides a simulation environment for training and evaluating dexterous hand models.
*   Includes geometric feature extraction and spatial relationship modeling between hand and object.
*   Supports training on diverse manipulation tasks with visual and proprioceptive inputs.

## Tech Stack
*   Python
*   PyTorch
*   NVIDIA Isaac Gym / Isaac Sim

## Getting Started
1.  Clone the repository: `git clone https://github.com/zoreanuj/dexterous-manipulation-sim.git`
2.  Install dependencies: `pip install -r requirements.txt`
3.  Follow the instructions in the original source repository for environment setup and training.
